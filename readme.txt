== Description ==
Umožňuje nahrávat texty (i s obrázky) zakoupené na Copywriting.cz přímo na Váš web bez nutnosti kopírování. Dále také automatické publikování objednaných PRčlánků.

== Installation ==
Instalace je velmi snadná. Více níže:

e.g.

1. Nahrejte plugin na Váš web (buď ručně do /wp-content/plugins/ nebo automaticky přes administraci)
2. Aktivujte jej
3. Jděte na copywriting.cz, vyberte web a tam dejte exportovat do Wordpressu, zadejte adresu Vašeho webu a dejte přidat

Web se tímto spáruje s Copywriting.cz a příště už web vyberete z roletky.
